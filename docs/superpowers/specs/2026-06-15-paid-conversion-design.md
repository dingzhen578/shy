# 历史学姐笔记付费转化优化设计

## 目标

在不接入真实支付、登录或数据库的前提下，让试用用户更清楚体验版能解决什么问题，并继续使用现有会员码完成开通。

## 产品规则

- 普通用户每个浏览器每天免费生成 1 次。
- 生成成功后扣除次数，生成失败不扣除。
- 免费用户仅可查看基础生成结果。
- 复制答案和保存学习卡片图仅对有效会员开放。
- 7 天会员、月卡和永久测试会员沿用现有会员码逻辑。
- 不新增支付、订单、登录、数据库或后台系统。

## 页面信息顺序

移动端按以下顺序展示：

1. 产品标题和每日免费次数。
2. 题目输入区和示例题目。
3. “适合这些场景”区域。
4. AI 答案学习笔记卡片。
5. 会员权益与价格开通卡片。
6. 会员码激活区域。
7. “大家一般这样用它”信任感区域。

桌面端保持当前最大宽度和单列阅读结构，不增加复杂导航。

## 使用场景区域

标题为“适合这些场景”，展示四个轻量小卡片：

- 月考前整理历史主观题
- 把错题整理成答题框架
- 快速复习背景 / 影响 / 意义
- 不想每次都从零组织答案

该区域放在输入区之后，让用户先理解产品用途，再看到答案和付费信息。

## 免费与会员能力

免费用户生成成功后仍显示“复制答案”和“保存学习卡片图”的入口，但按钮呈会员锁定状态。点击后不执行复制或导出，而是：

1. 显示页面内温柔提示。
2. 平滑定位到会员开通卡片。

会员用户使用两个功能时维持现有行为。

## 开通卡片

开通卡片对所有未激活会员的用户可见，不再只在免费次数耗尽后出现。免费次数耗尽时，标题显示“今天的免费次数用完啦 ✨”；仍有次数时，标题显示“解锁完整学习笔记功能 ✨”。

### 7 天体验版

价格：9.9 元 / 7 天

- 7 天不限次数生成
- 支持复制答案
- 支持保存学习卡片图
- 适合考前集中整理错题

### 月卡

价格：14.9 元 / 月

- 30 天不限次数生成
- 适合长期复习历史主观题

卡片保留“联系我开通”按钮和现有微信二维码。会员码激活表单保留，位置调整到开通权益之后。

## 信任感区域

标题为“大家一般这样用它”，展示：

- 整理错题
- 考前复习
- 练习主观题结构
- 生成学习卡片

内容使用简洁标签或小卡片，不编造用户数量、评价或成交数据。

## 视觉规范

- 延续奶油白、淡粉、淡紫和学习手账风格。
- 使用现有圆角、柔和边框和阴影，不引入新动画体系。
- 价格和核心权益使用玫瑰色强调。
- 手机端按钮高度不低于现有主按钮，卡片保持单列和足够行距。
- 不增加横向滚动。

## 状态与错误处理

- localStorage 中旧的每日剩余次数会通过现有归一化逻辑自动收敛到 1 次。
- 会员过期后继续沿用现有清理和提示逻辑。
- 免费用户点击受限功能时显示“开通体验版后，就可以复制和保存学习卡片啦 ✨”。
- 不使用浏览器 alert。

## 测试范围

- 每日免费上限为 1，跨日重置为 1，旧的 2 或 3 次数据被限制为 1。
- 免费用户不能执行复制和图片分享逻辑。
- 会员用户仍可复制和保存学习卡片。
- 页面包含全部场景、权益、价格和信任文案。
- 手机宽度下无横向滚动，输入、生成和开通按钮可正常点击。
- `npm test`、`npm run lint` 和 `npm run build` 全部通过。
